CSS 선택자

1.타입선택자 
가장기본적인 선택자  
ex) h1 {color : green; } 앞부분을 key 뒷부분을 value라 한다.

2. 전체 선택자
*{color : blue;}

3. ID 선택자
#special {color : red;}
<p id="special"> (p단락에 special의 스타일 시트가 붙게된다)
